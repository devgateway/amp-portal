
This directory should contain patches with a full list of changes made to Drupal
core, contrib modules or libraries.

Patch filename patern:
  YYYYMMDD--MODULE--REASON_TXT.patch

  * YYYYMMDD   - Date when the patch was applied.
  * MODULE     - Name of the module or library the changes apply to. Use CORE for the Drupal core.
  * REASON_TXT - A brief description of what the patch does. Use uderscore "_" as word separator.
