
This folder contains example update scripts with the following name format:
  update.XXXX.php

XXXX - is an incremental number or an environment name.

For example:
  p1.0/update.XXXX.php
  p2.0/update.XXXX.php
  p2.1/update.XXXX.php
  update.local.php
  update.staging.php
  update.production.php

Update scripts should be placed in sub-folders (useful for large projects).
